#  Spam Email Detection (Phishing Email Classifier)

This project is a Machine Learning-based system that detects whether an email is **Safe** or **Phishing** using Natural Language Processing (NLP) and Logistic Regression.

---

##  Project Overview

- Classifies emails into:
  -  Safe Email
  -  Phishing Email
- Uses TF-IDF for text vectorization
- Logistic Regression model for classification
- Includes preprocessing like stopwords removal

---

##  Dataset

Dataset used:
Phishing Emails Dataset

https://www.kaggle.com/datasets/subhajournal/phishingemails
